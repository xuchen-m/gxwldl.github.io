![](/img/塑料绝缘电缆.jpg)

# 全塑电力电缆
![](/img/全塑电力电缆.jpg)
```shell
一、适用范围

本产品适用于有较高防火和阻燃要求，交流额定电压0.6/1kV及以下系列的耐火电力电缆。

二、使用特性

1、聚氯乙烯绝缘导体的最高额定温度为70℃，交联聚乙烯绝缘电缆导体的最高额定温度为90℃。

```
# 低烟无卤电线
![](/img/低烟无卤电线.jpg)
```shell
一、适用范围

   本产品主要用于高层建筑、地下铁路、地下街、舰船。海上石油平台、大型电站及重要的工矿企业等与消防安全和消防救生有关的地方，以及燃烧时要求低烟无卤阻燃、耐火场所，作输送电能线路用.软结构的低烟无卤阻燃线缆，更适合于机房使用。

二、使用特性

1、电缆导体的最高额定温度为90℃。

2、短路时（最长持续时间不超过5S）电缆导体的最高额定温度不超过250℃。

3、电缆敷设时环境温度应不低于0℃，其最小弯曲半径单芯电缆为20D,多芯电缆为15D（D为电缆外径）
```
# 架空电缆
![](/img/架空电缆.jpg)
```shell
一、本产品适用于交流额定电压U0/U为0.6/1kV及以下架空电力电缆线路用铝芯聚乙烯（简称PE）绝缘架空电缆。可作为固定敷设用或引户线。

二、使用特性

1、电缆导体的长期允许工作温度：不超过70℃。

2、电缆的敷设温度应不低于-20℃。

3、短路时（最长持续时间5s）电缆允许的最高温度：聚乙烯绝缘为150℃.

4、电缆的允许弯曲半径：

     电缆外径（D）小于25mm者，应不小于4D。

     电缆外径（D）为25mm及以上者，应不小于6D。
```

# 钢芯铝绞线
![](/img/钢芯铝绞线.jpg)
```shell
1、相关结构

钢芯铝绞线是由铝线和钢线绞合而成的，适用于架空输电线路用。它内部是钢“芯”，外部是用铝线通过绞合方式缠绕在钢芯周围；钢芯主要起增加强度的作用，铝绞线主要起传送电能的作用；

2、相关型号

钢芯铝绞线的型号分为：JL/G1A  JL/G1B  JL/G2A  JL/G2B  JL/G3A。

其中：
G1A或G1B指的是普通强度钢线。

G2A或G2B 指的是高强度钢线。

G3A指的是特高强度钢线。

型号差异：以JL/G1A-240/30型钢芯铝绞线为例。

在国标（GB/T1179-2008）中以目前的JL/G1A-240/30表示；

在国标（GB/T1179-1983）中以LGJ-240/30表示。

3、主要应用

对高电压的输电线路，由于线路的路径有限，又不可能在一个路径上架多条线路，人们想出了采用一个路径，一套铁塔，挂多条导线，即一条线路相当于架了多条线路的方法；对500KV的输电线路，我国大多采用挂四条导线的方式，这种架线的方式称为同塔并架。

4、产品特点

钢芯铝绞线具有结构简单、架设与维护方便、线路造价低、传输容量大、又利于跨越江河和山谷等特殊地理条件的敷设、具有良好的导电性能和足够的机械强度、抗拉强度大、塔杆距离可放大等特点。因此广泛应用于各种电压等级的架空输配电线路中。

```
# 高压交联电缆
![](/img/高压交联电缆.jpg)
```shell
一、产品名称

　　6~35KV交联聚乙烯绝缘电力电缆

二、产品标准
　　本产品按GB/T12706《额定电压35KV及以下铜芯、铝芯塑料绝缘电力电缆》标准生产，同时还可根据用户需要按国际电工委员会推荐标准IEC、英国标准、德国标准及美国标准生产。

三、适用范围

　　本产品适用于工频额定电压3.6/6KV~26/35KV输配电线路作配送电能之用。

四、使用特性

　　工频额定电压U0/U为3.6/6KV~26/35KV。

　　电缆导体的允许长期最高工作温度为90℃。

　　短路时（最长持续时间不超过5s）电缆导体的最高温度不超过250℃。

　　电缆敷设时环境温度应不低于0℃。

　　电缆弯曲半径：三芯电缆不小于电缆外径15倍；

　　单芯电缆不小于电缆外径20倍。
```

# 中压交联电缆
![](/img/中压交联电缆.jpg)
```shell
一、产品特点：

    本产品适用于额定电压3.6/6～26/35kV输配电系统。

    电缆导体最高额定温度：90℃

    短路（最长时间持续5秒）电缆导体的最高温度不超过250℃。

    电缆敷设温度低于0℃，应预先加热。

    电缆安装时的电缆最小弯曲半径：无铠装单芯电缆20D，无铠装三芯电缆15D,有铠装单芯电缆15D，有铠装三芯电缆12D

二、执行标准

额定电压3.6/6～26/35kV交联聚乙烯绝缘电力电缆执行标准IEC 60502.2-2005，IEC 60840-2011，GB/T 12706-2008等标准要求。
```
